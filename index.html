A/B Sample Size Calculator

This tool will allow you to estimate required sample sizes for A/B tests that utilize the g-test statistic for significance.

Use the sliders to select the current conversion rate and what effect you expect the variant to have on this rate. Finally, specify what percentage of visitors will be allocated to each group and click 'Calculate' to determine the sample size you will need.

The tool will report the sample size needed in each group at the 5% significance level. It will also graph how these values might change for different sized treatment effects and different significance levels.
